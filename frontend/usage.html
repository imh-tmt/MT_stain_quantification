Step-by-step instructions in Korean and English for image upload, ROI calibration, and result export.

Korean

사용 방법

  1. 메인 화면에서 Masson's trichrome 염색 이미지를 업로드합니다. PNG, TIFF, JPEG 등 일반 이미지 형식을 사용할 수 있습니다.
  2. 업로드 후 자동으로 기본 stain vector를 사용한 collagen deconvolution 결과가 계산됩니다.
  3. 슬라이드별 색상 차이가 크면 Select ROIs를 누른 뒤 collagen, counter stain, background 영역을 각각 드래그해 선택합니다.
  4. ROI 선택 후 Process를 눌러 보정된 stain vector로 다시 분석합니다.
  5. Fibrotic Burden 값은 전체 조직 영역 중 collagen-positive pixel의 비율입니다.
  6. 결과 값 또는 복사 버튼을 클릭하면 Excel이나 통계 프로그램에 붙여넣을 수 있습니다.
  7. Collagen-only 이미지는 다운로드 버튼을 눌러 PNG 파일로 저장할 수 있습니다.

분석 일관성을 위해 한 실험 세트에서는 동일한 threshold와 ROI 선택 기준을 사용하세요.

English

How to Use

  1. Upload a Masson's trichrome-stained image from the main analysis page. Common image formats such as PNG, TIFF, and JPEG are supported.
  2. The app automatically performs collagen deconvolution using the default stain vectors after upload.
  3. If staining color varies across slides, click Select ROIs and draw representative collagen, counterstain, and background regions.
  4. Click Process to re-run the analysis using ROI-calibrated stain vectors.
  5. Fibrotic Burden represents the percentage of collagen-positive pixels within the valid tissue area.
  6. Click individual values or the copy button to paste the results into Excel or statistical software.
  7. Use the download button to save the collagen-only output image as a PNG file.

For consistent quantification, use the same threshold and ROI selection criteria within each experimental dataset.
