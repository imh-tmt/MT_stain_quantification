Step-by-step instructions in Korean and English for image upload, ROI calibration, and result export.

Korean

사용 방법

  1. 메인 화면에서 Masson's trichrome 염색 이미지를 업로드합니다. PNG, TIFF, JPEG 등 일반 이미지 형식을 사용할 수 있습니다.
  2. 업로드 후 자동으로 기본 stain vector를 사용한 collagen deconvolution 결과가 계산됩니다.
  3. Collagen Only 이미지 또는 Segmented Mask가 실제 염색 상태나 육안 분석과 다르게 보이면 Select ROIs를 누른 뒤 collagen, counter stain, background 영역을 각각 드래그해 선택합니다.
  4. ROI를 정확히 지정하려면 마우스 휠로 이미지를 확대하고, 마우스 드래그로 위치를 이동한 뒤 해당 stain이 뚜렷하게 염색된 부분만 선택합니다.
  5. ROI 선택 후 Process를 눌러 보정된 stain vector로 다시 분석합니다.
  6. Fibrotic Burden 값은 전체 조직 영역 중 collagen-positive pixel의 비율입니다.
  7. 결과 값 또는 복사 버튼을 클릭하면 Excel이나 통계 프로그램에 붙여넣을 수 있습니다.
  8. Collagen-only 이미지는 다운로드 버튼을 눌러 PNG 파일로 저장할 수 있습니다.

Usage tip: ROI는 넓게 잡기보다 해당 stain 색이 선명하고 다른 염색 또는 배경이 섞이지 않은 부분을 작고 정확하게 선택하는 것이 좋습니다.

English

How to Use

  1. Upload a Masson's trichrome-stained image from the main analysis page. Common image formats such as PNG, TIFF, and JPEG are supported.
  2. The app automatically performs collagen deconvolution using the default stain vectors after upload.
  3. If the Collagen Only image or Segmented Mask does not match the actual staining pattern or visual assessment, click Select ROIs and draw representative collagen, counterstain, and background regions.
  4. For accurate ROI selection, zoom in with the mouse wheel, pan by dragging the image, and select only areas where the target stain is clearly represented.
  5. Click Process to re-run the analysis using ROI-calibrated stain vectors.
  6. Fibrotic Burden represents the percentage of collagen-positive pixels within the valid tissue area.
  7. Click individual values or the copy button to paste the results into Excel or statistical software.
  8. Use the download button to save the collagen-only output image as a PNG file.

Usage tip: A small, clean ROI with a clearly stained target area is usually preferable to a large ROI mixed with other stains or background.
