Masson's Trichrome Analysis

Methods

Manuscript Text

Suggested Methods Paragraph

Masson's trichrome-stained tissue images were analyzed using a web-based color deconvolution tool implemented in Python/FastAPI and JavaScript. RGB images were converted to optical density (OD) space using OD = -log(I/I0), where I is the measured RGB intensity and I0 is the incident light/background intensity. When a background region of interest (ROI) was selected, I0 was estimated from the mean RGB intensity of that ROI; otherwise, normalized white illumination was used. Collagen and counterstain components were unmixed using a Ruifrok and Johnston color deconvolution approach with Masson's trichrome stain vectors. When the collagen-only output or segmented collagen mask did not match visual assessment of the stained tissue, collagen and counterstain vectors were recalibrated using user-selected ROIs. The collagen concentration channel was converted to an 8-bit transmittance-style grayscale image using 255 x exp(-C), where C is the collagen concentration. Collagen-positive pixels were segmented using a fixed grayscale threshold of 0-180. Tissue area was estimated by excluding bright background pixels from the original image using a mean RGB intensity threshold of 235. Fibrotic burden was calculated as: collagen-positive pixels within tissue / total tissue pixels x 100.

Algorithm

Calculation Summary

  1. Load the input image and convert it to RGB.
  2. Estimate background illumination from a selected background ROI or use normalized white illumination.
  3. Convert RGB intensities into optical density space.
  4. Separate stain contributions using the inverse stain matrix for collagen, counterstain, and residual/background components.
  5. Generate collagen-only RGB, 8-bit grayscale, and binary mask outputs.
  6. Exclude bright non-tissue background and compute collagen-positive area as a percentage of valid tissue area.
